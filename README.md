🏠 House Price Predictor (ML Project)

A simple and interactive Machine Learning web app that predicts house prices based on property features like location, house type, land area, building condition, and construction year.

✨ How this project was built

Collected and understood housing dataset
Cleaned and preprocessed the data
Converted categorical values into numerical format
Trained a Linear Regression model for prediction
Built an interactive Streamlit web app for users

🚀 Features

🏡 Predict house price instantly
📍 Select location and zoning type
📏 Enter land area (square feet)
🏠 Choose house type easily
🔧 Check building condition
📅 Select construction year
⚡ Fast and simple UI

🛠 Tools & Technologies

Python
Pandas & NumPy
Scikit-learn (Linear Regression)
Streamlit
Joblib
Git & GitHub

🎯 Project Goal

To automate house price prediction using Machine Learning
To understand real-world regression problems
To build a simple and user-friendly web application
